# learn-javascript-authentication

| Endpoint    | Method | Description                         |
| ----------- | ------ | ----------------------------------- |
| `/`         | GET    | Hello                               |
| `/register` | GET    | Register of users                   |
| `/login`    | POST   | Login with existing user            |
| `/users`    | GET    | Get all users                       |
| `/secret`   | GET    | Get profile based on logged in user |
